Muu Run

Muu Run é um jogo de infinity run (corrida infinita) conta a história de Muu, uma vaquinha comum que após alimentar-se se uma materia misteriosa ganha super poderes.

Após a descoberta dos poderes de Muu as autoridades locais lançam uma campanha de caça a vaquinha recompensando quem a capturar primeiro. Então Muu se vê obrigada a fugir dos fazendeiros que tentam captura-la a qualquer custo.

Ao longo do jogo você poderá acompanhar a história e descobrir os mistérios que cercam os poderes de Muu.
